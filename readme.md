# Laravel chuhe

## 安装laravel的方法
+ 检出代码到本地环境，[laravel](https://github.com/laravel/laravel).
+ 配置composer为国内镜像: `composer config repo.packagist composer https://packagist.phpcomposer.com`
+ 然后执行`composer install`
+ 然后执行 `php artisan key:generate生成key`
+ 本地配置虚机主机加伪静态环境
